# projection-domain-ct-enhancement
Implements a supervised U-Net–based framework for denoising CT projection data upstream of reconstruction. Includes code for training on paired low-/high-dose projections, CT image reconstruction, and inference with a discrete denoising factor to control enhancement strength.
